Attributes: 

            
    
                
  • cellspacing: 
    •  space between each cell.
                
  • cellpadding: 
    •  space between content and cell.
                
  • rowspan: 
    •  merging 2 or more rows.
                
  • colspan: 
    •  merging 2 or more columns.
                
  • bgcolor: 
    •  set a background color
                
  • align: 
    • set the content.
                
  • style: 
    • applying style.
                
  • border-colleapse: 
    • merges adjacent cell border into a single border.
                
  • border: 
    • border for the table.
                
  • height: 
    • height of the table.
                
  • width: 
    • width of the table.
