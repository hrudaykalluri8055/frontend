multimedia:

Adding different types of media like that image, audio, video, animations and interactive content.

Concepts:

  1. Image
  2. Audio
  3. Video
  4. Iframe

  • Images: 

       The image tag is used to embed images. it is a self closing tag(inline element).
                        src -> path of the image
                        alt -> Alternate text
                        Width/heigth -> Dimenstions of image.


  • Audio:

       It is used to embed sounds
                            src -> Path of the audio.
                            controls -> display play, pause, volume control
                            autoplay -> audio start automatically
                            loop -> Replay the audio 
                            muted -> Starts audio in muted.


  • Video: 

       It is used to embed video
                            src -> Path of the Video.
                            controls -> display play, pause, volume control, fullscreen.
                            autoplay -> play automatically
                            loop -> Repeat video 
                            muted -> Starts video without sound.
                            poster -> thumbnail image before the video plays.
                            width/height -> sdet the size.

  • Iframe: 

     It is used to embeded the other web page source in to the current web page source. 
                            src -> path of the another webpage
                            target -> link inside iframe opens.
                            frameborder -> shows or hides the border
